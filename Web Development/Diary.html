WEBサーバー・サイト作成のメモ

使用したWebサーバーの移り変わり

  1. Raspberry Pi4 Model B (RAM4GB)
  2. Oracle Cloudの無料枠
  3. Conoha VPSのメモリ1GBプラン

使用したOS

  • Raspberry Pi OS
  • Oracle Linux
  • Ubuntu 22.04

Raspberry Pi時代

  • とりあえずRaspberry Pi OSを入れてApacheを導入
  • 自宅内(ローカル)ネットワークでWebサーバー化
  • HTMLとCSSベタ打ちでページを作成
  • メモリ4GBモデルだったこともあり、かなり快適に動作していた
  • ARM系CPUだったこともあり、対応していない機能が多かった
  • ローカルからグローバルにすべく、自宅のポートを開放するために色々調べて勉強した
  • 当時、自覚していなかったが、非常に良くないポート開放のやり方をしていた
  • WEBページにパスワード認証をつけたかったが、よくわからなかったのでJavascriptでそれっぽいのを実装していた
    • 参考にしたURL[https://allabout.co.jp/gm/gc/23839/]
  • この頃既に、将来的にはConoha VPSを借りる予定を立てていたので、ドメインは取っていない
  • HTMLとCSSについて少しずつ理解し始める
  • 当時はこんな感じのデザイン
  • HTML/CSS全て手打ちなので色々学べた

Oracle Cloud時代

2月上旬に部活の先輩から「無料でサーバー借りれるところあるよ」と紹介されて使いはじめる
  • 初めてARM系CPU以外のサーバーを触った
  • 初めてUbuntuを入れた
  • 無料枠のメモリが1GBだからConohaで契約予定のプランと同じと思いどのような動作になるか試せると思った
  • DNSとかを実際に触って色々学べた
  • ラズパイはGUIで直接触っていたが、Oracleになってターミナル上からのみ操作ができるので色々戸惑った
  • 特にファイル権限(パーミッション)がなくてディレクトリをいじれなかった時は何事かと思って調べまくった
  • サーバーにVSCodeのSSH接続をするとリソースを食い散らかす問題が発生したが、VSCodeのファイル監視を停止することで解決した
    • 3個にしたURL[https://blog.masuyoshi.com]
  • デザインが綺麗になり始める

Conoha VPS時代

3/10に契約
  • 3月チュジュンからConoha VPSのメモリ1GBプランを借りてWEBサーバー化を進める
  • Cloudflareでドメインを購入してSSL証明証も発行(?)
  • 初めて自分の作ったWEBサイトに「安全な通信です」と表示されて嬉しかった記憶
  • Oracleの時と違ってCPUがAMD→Intelになった影響かVSCodeでSSH接続してファイル監視を停止させなくても落ちなくなった
  • 先輩も言っていたが、AMDCPUのサーバーはリソース管理が？な気がする
  • あらゆる動作が快適になって作業効率化
  • この頃にWordPressの導入を決定
  • WordPressを導入するにあったってデータベース(MariaDB)を導入する
  • が、「データベース接続確立エラー」と出て一向に進まない
  • MariaDBでユーザーを作るにあたって「Access denied for user 'root'@'localhost’」と出る
  • 正しく書いているつもりだが、「' 」「’」が混ざっていたことが判明
    • 直したらできた
  • WEBサイトは完全にログイン制にして、ログインする人によって違うページを表示させたかったので、function.phpにスクリプトを追加
    • 参考URL[https://hirashimatakumi.com/blog/3713.html]

通して

毎回ファイル/ディレクトリのパーミッションがしっかりと設定されてなくて詰まっていたことがあった 「SSHホスト鍵が変わってるよ!」と言われたりrootでのSSH接続を禁止したり基本的なことはある程度体験できた

最後に

  • ただ、セキュリティ関係がまだ怖い
  • 攻撃に晒されてアクセスできなくなるのも怖いけど不正アクセスされてデータを持ってかれていくとこの方が怖い
  • 攻撃・不正アクセスされた時に、自分がしっかりとタオ言うできるかわからないのが怖い
  • 先輩にサーバーを攻撃してもらった時はCloudflareによって弾かれたからよかったが、人為的な脆弱性があったらどうしようもない
